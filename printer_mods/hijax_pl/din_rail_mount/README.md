# README

### Overview

This is a mount for DIN RAIL in legacy v2.2 way.
Angled version can be used to attach DIN rail to the back of Voron.

### Print parameters

as per Voron specifications:

- Layer height: 0.2
- Infill: 50% or more
- Infill type: Grid
- Perimeters: 4 or more
- Top/Bottom solid layers: 5 or more
- Filament: ABS/ABS+
