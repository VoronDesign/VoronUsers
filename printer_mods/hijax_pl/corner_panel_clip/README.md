# README

### Overview

This is a corner modification to alllow installing panels when some thoer way of extruction connection is used.

### Print parameters

as per Voron specifications:

- Layer height: 0.2
- Infill: 50% or more
- Infill type: Grid
- Perimeters: 4 or more
- Top/Bottom solid layers: 5 or more
- Filament: ABS/ABS+
