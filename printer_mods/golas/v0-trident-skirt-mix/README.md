* stl files with print number suffix

## BOM
In addition to preloaded nuts and screw that are in default v0.1 version you need also:
* 4 x standard v0.1 BOM heat set inserts
* 6 x M3x12 BHCS
* 2 x M3x6 BHCS
* BTT MINI 12864 V1.0
