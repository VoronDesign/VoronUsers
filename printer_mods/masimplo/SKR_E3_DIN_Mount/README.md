# SKR E3 Family DIN mount

This mod can be used with the SKR E3 family boards and can be mounted on a DIN rail using a Voron DIN clamp.

Printed Parts:
- 1x SKR E3 Mount
- 1x Standard Voron din clamp

Hardware:
- 4x M3 Heat inserts
- 4x M3 x 6 BHCS
- 2x M2 self tapers
